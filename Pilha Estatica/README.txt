Algoritmo para converter um número decimal em binário implementado usando PILHAuso.
Esse é um exemplo de uso da estrutura de pilha.

Obs. Nesse algoritmo, foi implementada todas as funções de pilha, inclusive a leNo ( ) que não foi usada.

Pilha
Em uma pilha, os itens são colocados uns sobre os outros, portanto, o último item inserido está no topo e o primeiro item inserido está na base. O modelo intuitivo de uma pilha é um monte de pratos em uma prateleira.  
É um caso particular de lista no qual as operações de inclusão e exclusão são realizadas na mesma extremidade denominada Topo.  

Estratégia LIFO (last in first out).

Quando utilizar? Quando for necessário inverter a ordem de processamento. Quando houver necessidade de que as estruturas mais internas sejam processadas antes das mais externas.
Um exemplo é um algoritmo para converter um número decimal para binário, já que os restos da divisão que irão compor o número binário, serão montados de trás para frente.
