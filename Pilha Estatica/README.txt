O arquivo dec_bin_pilha.c contem o código de um algoritmo para converter um número decimal em binário, implementado usando PILHA.
Esse é um exemplo de uso da estrutura de dados "pilha".

O arquivo pilha_estatica.c é um programa com um menu de opções para construir uma pilha, empilhar elementos (PUSH), desempilhar elementos (POP), exibir, esvaziar, enfim, as funções de processamento de pilha.

------------------------------------------------------------------------------------------------------------------------------------------------

PILHA

Em uma pilha, os itens são colocados uns sobre os outros, portanto, o último item inserido está no topo e o primeiro item inserido está na base. O modelo intuitivo de uma pilha é um monte de pratos em uma prateleira.  
É um caso particular de lista no qual as operações de inclusão e exclusão são realizadas na mesma extremidade denominada "topo".  

Estratégia LIFO (last in first out).

Quando utilizar? Quando for necessário inverter a ordem de processamento, no caso do algoritmo dec_bin_pilha.c, os restos das divisões são empilhados pela execução do algoritmo, depois desempilhados para compor o número binário como saída. 
Sempre que houver necessidade de que as estruturas mais internas sejam processadas antes das mais externas, devemos utilizar pilha.
