Exemplo didático de código em C utilizando Strings e algumas funções da biblioteca string.h

É um programa console que captura o nome do usuário e o sexo, imprime uma mensagen de saudação Sr.(a) seguido do nome digitado, pedindo para digitar a senha, depois verifca se a senha digitada é igual a senha definida no código, que nesse caso é a string: "provaav1".

Se o usuário digitar provaav1, a senha estará correta e o programa imprimirá uma mensagem:
 "Senha Correta"
 "Você digitou a senha"
 "p r o v a a v 1"

Caso contrário, ele perderá uma chance do total de chances que é 3.

O programa imprimirá:
Senha Errada!!!
Você tem mais 2 chances!!!

Se errar de novo:
Senha Errada!!!
Você tem mais 1 chances!!!

Se errar a última chance, o programa imprimirá: 
"Chances Esgotadas!!!" 

E finalizará.
